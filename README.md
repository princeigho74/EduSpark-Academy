## Installation

1. Clone the repository:
```bash
git clone https://github.com/princeigho74/eduspark-online-academy.git
cd eduspark-online-academy
```

2. Install dependencies:
```bash
npm install
```

3. Start development server:
```bash
npm start
```

4. Build for production:
```bash
npm run build
```

## Deployment

### Deploy to Vercel

1. Push to GitHub
2. Import project on Vercel
3. Deploy automatically

Or use Vercel CLI:
```bash
npm install -g vercel
vercel
```
